# Emenu - Django & VueJs 



## Setup Django app

1. Clone repository
2. Create virtualenv
2. Install Django dependencies

```sh
$ pip install -r requirements.txt

```


### Run the Application

```sh
$ python emenu/manage.py runserver 0.0.0.0:8000
```

### Testing

```sh
$ python emenu/manage.py test
```

## Setup Vue APP
```sh
$ cd emenu-client
$ yarn install
```
## Run Vue APP
```sh
$ yarn run serve
```
